# Recluze

**Get a WhatsApp bot running on your number in under 5 minutes.**

Recluze is a standalone desktop app that turns your personal WhatsApp into an AI-powered bot. Pick a persona (Support, Booking, Resume, Personal, Services), add an LLM provider key, scan a QR code, done.

No Python. No Node. No Docker. No cloud account. Download the installer, double-click it, and you have a bot.

## Quick start

### 1. Download

Grab the installer for Windows from the [latest release](https://github.com/MuhammadMustafa18/Recluze/releases/latest):

- **`Recluze_0.1.0_x64-setup.exe`** — 83 MB NSIS installer (recommended, no admin needed)
- **`Recluze_0.1.0_x64_en-US.msi`** — 98 MB MSI installer (per-machine)

### 2. Install

Double-click the `.exe`. Standard Next → Install → Finish. On Windows 10 the installer offers to fetch WebView2; on Windows 11 it's already there.

### 3. Add an LLM provider

Launch Recluze from the Start Menu. Open the **Providers** tab → click one of the presets (Groq, OpenAI, Claude, Ollama, LM Studio, or Custom) → paste your API key → Save.

A free Groq account gets you thousands of messages per day at the time of writing.

### 4. Pick a persona

Open the **Personas** tab → click one of the presets (Support, Booking, Resume, Personal, Services) → Save. Edit the system prompt inline, or upload a PDF knowledge base.

### 5. Scan the QR

Open the **Connections** tab → scan the QR with your phone (WhatsApp → Linked Devices → Link a Device).

### 6. Send yourself a message

Your bot is live. Every message you send to your number gets a reply generated by the active persona + provider.

**Total time: under 5 minutes.**

## What you get

| Feature | Description |
|---|---|
| **Persona presets** | Support, Booking, Resume, Personal, Services — each with a tuned system prompt. Pick one and ship. |
| **Custom personas** | Hit "Custom" to clear the preset and write your own from scratch. |
| **PDF knowledge base** | Upload a PDF per persona. Text is extracted and injected into every reply. |
| **Provider presets** | GPT-4o, GPT-4o Mini, Claude, LM Studio, Ollama, Groq. Custom reset for anything else. |
| **Local LLMs** | Works with Ollama and LM Studio out of the box — your messages never leave your machine. |
| **Live logs** | Observability tab streams structured logs in real time, with level filters and request-ID tracing. |
| **Multi-device WhatsApp** | Built on Baileys — your phone stays the source of truth, the bot acts as a linked device. |

## Architecture

```
┌─────────────────────────────────────────────┐
│           Recluze.exe (Tauri / Rust)         │
│  ┌────────────┐  ┌────────────────────────┐  │
│  │  WebView2  │  │  Sidecar Manager       │  │
│  │  (React UI)│  │                        │  │
│  └────────────┘  └────────────────────────┘  │
└─────────────┬───────────────────┬────────────┘
              │ spawns            │ spawns
              ▼                   ▼
   ┌─────────────────────┐  ┌──────────────────────┐
   │  recluze-api.exe    │  │ baileys-sidecar.exe  │
   │  (PyInstaller)      │  │ (bun-compiled)       │
   │                     │  │                      │
   │  FastAPI :18234     │  │ WhatsApp protocol    │
   │  LangGraph          │  │ :2786 / :2787        │
   │  SQLAlchemy +       │  │                      │
   │  Alembic (SQLite)   │  │                      │
   └─────────────────────┘  └──────────────────────┘
```

All three binaries ship in a single installer. First launch copies the sidecars from `%LOCALAPPDATA%` to your per-user app data dir and starts them automatically. You never see a terminal.

## Where your data lives

| What | Where |
|---|---|
| SQLite database | `%APPDATA%\com.recluze.desktop\data\app.sqlite` |
| WhatsApp auth | `%APPDATA%\com.recluze.desktop\data\baileys-auth\` |
| Logs | `%APPDATA%\com.recluze.desktop\logs\` |

To reset: delete the `com.recluze.desktop` folder. The app will recreate it on next launch.

## Building from source

Requires Rust, Node 20+, Python 3.12+, and Bun.

```bat
:: Full pipeline (Baileys compile → PyInstaller → Tauri installer)
scripts\build_release.bat
```

Outputs:
- `desktop\src-tauri\target\release\bundle\msi\Recluze_0.1.0_x64_en-US.msi`
- `desktop\src-tauri\target\release\bundle\nsis\Recluze_0.1.0_x64-setup.exe`

## Development mode

```bash
# Backend (FastAPI + LangGraph)
python -m venv .venv && .venv\Scripts\activate
pip install -e .

# Frontend (React + Vite)
cd desktop && npm install && npm run dev

# Baileys sidecar
cd baileys-sidecar && npm install && npm run dev
```

In dev mode, run `npm run tauri dev` from `desktop/` to launch the Tauri shell with hot reload.

## Tech stack

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS + React Router v6
- **Backend:** FastAPI + SQLAlchemy + aiosqlite + LangGraph + structlog
- **WhatsApp:** Baileys (Node.js, multi-device protocol)
- **Desktop shell:** Tauri 2 (Rust + WebView2)
- **Packaging:** PyInstaller (Python) + Bun compile (Node) + WiX + NSIS (Tauri)
